## Quality Metrics

### Posting Threshold Checks
Every potential forum post must pass 6/8 checks:

1. **Has Data**: Minimum 5 data points
2. **Is Novel**: Not duplicate/similar to recent posts
3. **Is Relevant**: Relevance score > 0.7
4. **Answers Question**: Solves an identified issue
5. **Provides Action**: Actionable recommendations
6. **Not Too Frequent**: Min 1 hour between posts
7. **Daily Limit**: Max 5 posts per day
8. **Engagement Potential**: Predicted upvotes > 0.6

### Decision Framework
```javascript
if (qualityScore >= 6/8 && humanOverride === false) {
  executeAction();
  logDecision();
  monitorOutcome();
}
```
